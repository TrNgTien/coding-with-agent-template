# Tasks — agent rules

**Put:** actionable items: backlog, next, in-progress. Checkboxes OK. One line per task when possible.

**Reminder plugin:** due dates on tasks as `(@YYYY-MM-DD)` or `(@YYYY-MM-DD HH:mm)` — see [[Tasks/Intercity deadlines]] for example + settings checklist.

**Don’t put:** long narratives (→ `Daily/`), design docs (→ `Projects/`).

**Read when:** “what’s next?”, prioritization, standup-style summaries.
