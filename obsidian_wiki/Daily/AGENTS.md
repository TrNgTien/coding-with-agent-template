# Daily — agent rules

**Put (journal):** `YYYY-MM-DD.md` at **Daily root** (or one `Daily log.md`): focus, done, decisions, blockers, carry-over. Token-tight bullets.

**Put (vault changelog):** `Daily/logs/<theme>/YYYY-MM-DD[-slug].md` — vault structure, Obsidian, project-doc imports, **security** policy. Hub [[Daily/logs/Index]]; rules [[Daily/logs/AGENTS]]. No secrets in log bodies.

**Agents editing vault files:** Must follow root [[AGENTS]] **Obsidian logging (mandatory)** — journal **Done**, themed `Daily/logs/` when a theme fits, [[Daily/logs/Index]] for new log files, chat summary; skill **obsidian-vault-log** ([[Skills/Skills hub]]).

**Don’t put:** durable standards (→ `Context/`), full specs (→ `Projects/`).

**Read when:** “what happened lately?” → journal; “what changed in the vault?” → `Daily/logs/`.

**Scope:** One **date** in the question → read **that** `YYYY-MM-DD.md` first; expand only via wikilinks from it.
